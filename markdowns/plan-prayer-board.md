# 26청년부 기도함 웹앱 기획

## 목표
교회 청년부가 서로의 기도제목을 나누고, 함께 기도할 수 있는 모바일 친화적 웹앱.
스티커보드처럼 따뜻하고 편안한 느낌의 UI/UX.

---

## 핵심 기능

### 1. 기도제목 작성
- 익명 / 실명 선택 가능 (토글 or 이름 필드 비우면 익명)
- 기도제목 내용 입력 (텍스트)
- 작성 후 카드 형태로 보드에 추가됨

### 2. 기도 보드 (메인 화면)
- 스티커보드/포스트잇 느낌의 카드 레이아웃
- 카드마다: 이름(또는 "익명"), 기도제목 내용, 작성일
- 최신순 정렬
- 모바일에서 한 열, 데스크탑에서 2~3열 그리드

### 3. 🙏 기도했어요 반응
- 각 카드에 🙏 버튼
- 누르면 카운트 +1 (중복 가능 여부는 아래 결정사항 참고)
- "3명이 기도했어요" 같은 표시

---

## 기술 스택

| 구분 | 선택 | 이유 |
|------|------|------|
| 프론트엔드 | **React + Vite** | 빠른 개발, 컴포넌트 기반 |
| 스타일링 | **Tailwind CSS** | 유틸리티 기반으로 빠른 UI 구현 |
| 백엔드/DB | **Firebase (Firestore)** | 무료 티어 충분, 서버 관리 불필요 |
| 호스팅 | **Firebase Hosting** | 배포 간단, HTTPS 자동 |
| 상태관리 | 불필요 | Firebase 실시간 리스너로 충분 |

## 데이터 구조 (Firestore)

```
prayers (collection)
  └─ {docId} (document)
      ├─ name: string          // 이름 또는 "" (익명)
      ├─ content: string       // 기도제목 내용
      ├─ color: string         // 카드 색상 (예: "pink", "yellow", "purple", "mint", "blue")
      ├─ prayCount: number     // 🙏 카운트
      ├─ createdAt: timestamp  // 작성 시간
```

## UI/UX 방향

- **따뜻한 색감**: 크림/베이지 배경, 파스텔 톤 카드 (연분홍, 연노랑, 연보라, 민트, 연파랑 중 직접 선택)
- **포스트잇/스티커 느낌**: 카드에 살짝 기울기(rotate), 부드러운 그림자
- **폰트**: 손글씨 느낌 or 둥근 고딕 (예: Pretendard, 나눔손글씨)
- **애니메이션**: 카드 등장 시 부드러운 fade-in, 🙏 누를 때 작은 바운스

## 리스크 & 고려사항

- **스팸/악용**: 관리 기능 없으므로, 도배 시 대응 어려움 → 필요 시 작성 쿨다운(예: 1분) 추가 가능
- **Firebase 무료 한도**: 일 읽기 5만, 쓰기 2만 → 청년부 규모면 충분
- **개인정보**: 실명 선택 시 이름만 저장, 별도 개인정보 수집 없음
- **브라우저 지원**: 모바일 Safari/Chrome 우선 대응
